# ConvertTo-TitleCase

## SYNOPSIS
Convert a string to title case.

## SYNTAX

```
ConvertTo-TitleCase [[-String] <String[]>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Convert a string to title case.

## EXAMPLES

### EXAMPLE 1
```
ConvertTo-TitleCase -String 'my string'
```

Returns the string \`My String\`.
